# 화면 디자인 – 디자인모드, text 모드

설명:
> Android Studio는 화면(Activity)을 디자인하는 툴을 제공한다. 그리고 그 툴을 이용하여 java 코딩없이 화면을 쉽게 구현할 수 있다. 디자인한 결과는 XML 형식의 파일로 저장된다.
